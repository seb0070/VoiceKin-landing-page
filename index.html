딥보이스 기술의 발전으로 단 5초의 음성 샘플만으로도 완벽한 목소리 복제가 가능해졌습니다.

🎙️

5초면 충분한 클로닝

최신 AI 기술은 5초 분량의 음성만으로 특정인의 목소리를 완벽히 복제합니다. SNS에 올린 짧은 영상 하나로도 당신의 목소리가 복제될 수 있어요.

📞

가족 사칭 범죄

"엄마 나 사고났어. 돈 급히 필요해." 낯선 번호로 걸려오는 전화, 그런데 목소리는 아들이다. 가족 사칭형 피해는 전체의 33.7%를 차지하며, 피해자가 의심을 갖기 가장 어려운 유형입니다.

🧠

인간의 귀는 한계가 있다

인간의 청각은 음성의 미세한 주파수 패턴 차이를 구별하도록 설계되지 않았습니다. 아무리 주의해도 딥보이스를 실제 음성과 구별하기는 거의 불가능해요.

📈

급증하는 피해

2023년까지 이어지던 감소세가 2024년을 기점으로 급격히 반전됐습니다. 50대 이상 피해자 비중이 매년 높아지고 있으며, 기존 예방 교육만으로는 부족합니다.

주요 기능

Voicekin의 핵심 기능들

기능 1

자녀 앱
음성 등록 · 관리

  • 15문장을 낭독해 Voiceprint 생성이후 언제든 재녹음 · 삭제 가능
  • 원본 음성은 즉시 삭제벡터만 동기화, 개인정보 보호
  • 정질 · 자진 · 보호자 등 상태 표시
기능 2

부모 앱
온보딩 · 보호 시작

  • QR 1회 스캔 또는 카카오톡 딥링크로 자녀 계정에 연결
  • 앱을 열지 않아도 백그라운드에서 모든 통화를 감지
  • 시니어 친화 UI권한 허용 순차 진행 (녹음 파일 읽기 → 알림 → 배터리 제외)
기능 3

실시간 통화 감지
분석 · 경고

  • 85점 이상정상 판정, 알림 없이 백그라운드 저장
  • 50 ~ 85점주의 알림 발송, 재통화 버튼 제공
  • 50점 이하위험 알림 발송, 112 신고 · 1332 금감원 버튼 노출
기능 4

통화 종료 후
분석 · 보호 모드

  • 통화 종료 시 자동 트리거 발생별도 조작 없이 백그라운드에서 자동 처리
  • ECAPA + AASIS로 10초 이내 신뢰도 산출화자 인식 + 딥페이크 탐지 모델 병렬 분석
  • 차단이 아닌 경고만, 판단의 주체는 사용자오탐지로 인한 불편 최소화
기술

🧬

Voiceprint 등록

가족의 목소리에서 고유한 음성 지문을 추출해 저장합니다. 사람의 귀로는 구별할 수 없는 미세한 패턴까지 분석해요.

⚡

실시간 비교 분석

통화 중 들어오는 음성을 등록된 Voiceprint와 실시간으로 비교합니다. Cosine Similarity 기반으로 유사도를 정밀 측정해요.

🛡️

딥페이크 탐지 (개발 중)

화자 인식을 넘어 AI 합성 음성 자체의 아티팩트를 탐지하는 AASIST 모델을 추가 개발 중입니다.

개발 로드맵

지금 이 순간에도 개발 중입니다

1차
✓ 완료

두 음성 파일 직접 비교

SpeechBrain ECAPA-TDNN 모델로 두 음성 파일의 화자가 동일인인지 판별합니다. Similarity 점수와 is_same_speaker 결과를 반환합니다.

2차
진행 중

가족 Voiceprint 등록 · 저장 · 조회

가족 구성원별 음성 지문을 등록하고 관리하는 시스템. 여러 가족의 Voiceprint를 저장하고 개별 조회할 수 있습니다.

3차
예정

통화 음성을 등록 가족 전체와 비교

실시간 통화 음성을 등록된 모든 가족의 Voiceprint와 자동 비교합니다. 가장 유사한 가족을 식별하고 의심 통화를 탐지합니다.

4차
예정

딥페이크 탐지 모델 추가

화자 인식 외에도 AI 합성 음성 고유의 아티팩트를 탐지하는 AASIST 모델을 통합합니다. 탐지 정확도를 대폭 향상시킵니다.

5차
예정

Android 앱 연동

모든 기능을 Android 앱에 탑재합니다. 실시간 통화 모니터링, 단계별 경고 알림, 가족 Voiceprint 관리 기능을 제공하는 완성된 서비스를 출시합니다.
